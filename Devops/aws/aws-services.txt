Apache airflow commonly used for tasks like ETL pipelines , Machine Learning pipelines and automating Devops tasks.

Amazon managed workflow will manage the apache airflow workflows.

it take care of setup , scaling , security , upgrades and monitoring.

it contains collections of dags , direct ascilic graph .

a DAG is a collection of tasks .

to create a task we need operators - bash / python / Email 
to run a task we need executors - sequential , celery , local 

airflow will create on the VPC, we need to select 2 subnets for the high availability
